CTSA : Time Series Analysis and ARIMA Modeling in C

CTSA is a C software package for univariate time series analysis. Includes ARIMA and Seasonal ARIMA models. This library has been used in CISCO Connected Streaming Analytics 1.X and 2.X. https://manualsbrain.com/en/manuals/1689569/?page=118
