Software

There are a number of wavelet software implementations available across different platforms. I have mostly worked on Matlab,C++ and Java so these are the packages that I will mention on this page.

Wavelet Matlab packages

  • Matlab Wavelet Toolbox : Commercial Toolbox
  • Wavelab: Free package that you can build on.
  • Rice Wavelet Toolbox: It is free as well and contains more optimized codes.
  • Ivan Selesnick's software page:Matlab codes [ 2D, 3D and complex wavelets].

Wavelet C++/Java Packages

  • ImageLib : Pretty Basic C++ library.
  • BlitzWave: This library is based on Wim Swelden's lifting scheme.
  • Wave++ : I believe it is only available for unix systems.
  • Ian Kaplan's Wavelets in Java: Ian has also implemented wavelet algorithms in C++
