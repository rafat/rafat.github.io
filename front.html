WAVELIB

An ANSI C implementation of decimated and undecimated 1D Fast Discrete Wavelet Transforms. Decimated, Stationary and Maximal Overlap Transforms are included.Added : DWPT.

Online Demo More

CTSA

CTSA is a C software package for univariate time series analysis. ARIMA and Seasonal ARIMA models have been added as of 10/30/2014

More

OPTIMC

OptimC is a C software package to minimize any unconstrained multivariable function.Includes Nelder-Mead,Conjugate Gradient, BFGS, Levenberg-Marquardt and more methods.

More

HSFFT

A simple, straightforward FFT Implementation in C/C++. It is designed to give good performance for non-power of length 2 FFTs. Only 1D , unit-stride implementation as of now.

More

DYADWAVES

A Simple Graphical User Interface for Discrete Wavelet Transform build using C++/QT4. Binaries for Win32 and Linux available. Useful for academic applications.

More

CREGRES

Linear Regression Analysis in ANSI C. Includes Multiple Regression, Polynomial Regression and some simple statistical and probability distribution functions.

More

LIFTWAVE

This C++ Wavelet library implements lifting scheme based on Sweldens/Daubechies paper. This is a header file based out-of-place implementation of 1D and 2D LWT/ILWT.

More

WAVELET1D

C++ 1D/2D DWT Implementation for Win32 and Linux. Contains Binaries for 32bit architectures and is dependent on FFTW liibrary. Useful for academic applications.

More

WAVEPY

Discrete Wavelet Transforms fully programmed in Python+Numpy. 1D/2D Decimated and Stationary Wavelet Transform Implementations useful for academic applications.

More

WAVELIB ONLINE GUI

Wavelib library running in a browser using Emscripten. The website is built using AngularJS, Bootstrap and plotly.js.Data Input is enabled. View report for the outputs.
